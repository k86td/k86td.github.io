# ResourceQuota
Cette ressource permet de limiter l'utilisation de ressource dans un namespace. Par exemple:
```yaml
apiVersion: v1
kind: ResourceQuota
metadata:
	name: storagequota
spec:
	hard:
		persistentvolumeclaims: "10"
		requests.storage: "500Mi"
```

On peut voir de l'information sur les ressources actuellement utilis√©es dans un namespace en faisant `kubectl describe ns <namespace-name>`.
